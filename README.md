mrbox
====

## Descriotion

mruby binary manager

## sample
```
$mrbox -n sample mruby -- --verbose sample.rb
```

you need more infomation.
`mrbox help`
## Requirement
git  
rm  

support mac, linux

## Install
```
$docker-compose run compile

```
